# vue3-component-template

## 安装包
```
yarn 
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```
## 目录结构
```
    examples 文件用于测试
    src 用于编写组件
      --compName  组件名称
```
